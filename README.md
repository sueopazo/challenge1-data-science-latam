## 🎯 Resumen

Estudio integral de datos para **Alura Store** que evalúa 4 sucursales y determina cuál debería venderse para optimizar el negocio. Utiliza **métricas financieras y operacionales** para generar una recomendación basada en evidencia.

### Objetivo
Identificar la sucursal con menor rendimiento mediante análisis de datos y visualizaciones.

---

## 🔍 Funcionalidades

**6 Análisis Principales:**

1. **💰 Estudio de Ingresos** - Ganancias totales y participación porcentual
2. **📦 Ventas por Categoría** - TOP 3 sectores más comercializados
3. **⭐ Satisfacción del Cliente** - Calificaciones promedio (1-5)
4. **🏆 Productos Destacados** - Bestsellers y artículos de baja rotación
5. **🚚 Costos de Envío** - Eficiencia en distribución
6. **📋 Informe Estratégico** - Recomendación final fundamentada

**Visualizaciones:**
- Gráfico circular de ingresos
- Barras agrupadas por categorías
- Comparación de calificaciones
- Análisis de costos operativos
